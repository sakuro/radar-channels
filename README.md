# Radar Channels

A Factorio MOD that lists universe-mode radars grouped by channel.

## Features

- Opens a Radar Channel List window via toolbar button or keyboard shortcut (unbound by default).
- Lists universe-mode radars grouped by channel.
- Hover over a radar icon to preview its location via camera view.
- Click a radar icon to close the window and switch to remote view at that radar's location.
- Only radars belonging to the player's own force are shown.
